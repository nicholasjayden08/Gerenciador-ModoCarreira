# ⚽ Career Manager (Java)

A personal Java project that simulates a football club career management system in a terminal-based application.

The main goal of this project is to practice Object-Oriented Programming (OOP), logic building, and data manipulation using pure Java, without frameworks.

The project is being developed incrementally as part of a learning process.

---

## 🎯 Purpose

Simulate a simple football club management environment where the user can manage a club, sign players, sell players, and control the squad and budget.

---

## 🧠 Key concepts practiced

- Object-Oriented Programming (OOP)
- Encapsulation
- ArrayList manipulation
- Console-based menu system
- Separation of responsibilities
- Business logic design

---

## ⚙️ Current features

- Create a club with name and initial budget
- Sign players (with budget validation)
- Sell players (with budget update)
- List squad players
- Dynamic budget updates based on transfers

---

---

## 🚀 How to run

1. Clone the repository
2. Open the project in IntelliJ IDEA (or any Java IDE)
3. Run the `Main` class
